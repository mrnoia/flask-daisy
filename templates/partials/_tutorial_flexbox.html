2. Flexbox Layouts

Create flexible, responsive layouts using Flexbox

Flex Row (Default)

Item 1
Item 2
Item 3
<div class="flex gap-4">
  <div class="flex-1">Item 1</div>
  <div class="flex-1">Item 2</div>
  <div class="flex-1">Item 3</div>
</div>
What it does: flex enables flexbox. flex-1 makes items grow equally. gap-4 adds spacing.

Flex Column (Vertical)

Item 1
Item 2
Item 3
<div class="flex flex-col gap-3">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
